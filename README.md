# 网易教育产品部
> 该页面为练习前端开发的静态页面 [**预览成品请戳这里**](http://www.bx1987.com/edu/)

### 该单页应用的主要功能点如下：
1. 关闭顶部通知条，cookie记录
2. 关注“网易教育产品部”，cookie记录(`模拟，所以没有去做登陆`)
3. 顶部导航hover等
4. 轮播图（`淡入淡出，悬停停止动画，定时触发动画`）
5. 主内容区tab切换
6. 课程详情json数据处理
7. 机构介绍（`遮罩弹窗视频处理`）
8. 右侧热门推荐滚动处理
9. 页面布局动态适应（`只处理了小于1200px的状态，简化模拟响应式布局`）
10. 页面顶部进度条
